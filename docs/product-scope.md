# Pantiku MVP Product Scope

## Personas

- Donor: discover and support impactful campaigns/products.
- Orphanage Manager: manage orphanage profile, campaigns, products, and anonymized children data.
- Admin: governance, moderation, and oversight across all records.
- Volunteer: authenticated user with future contribution workflows.

## MVP Goals

- Enable transparent fundraising and product sales.
- Provide structured impact visibility for trust.
- Protect child privacy in all public-facing flows.

## MVP v0.1 Launch Scope

### Must-have

- Register/Login
- Role-based user
- Orphanage profile
- Campaign list/detail
- Donation intent
- Product marketplace list/detail
- Basic order intent
- Impact summary
- Admin seed user

### Nice-to-have later

- Real payment integration
- Real shipping integration
- Chat/forum
- Foster parent system
- ML matching
- Volunteer scheduling
- Mobile push notification

## Out of Scope (MVP)

- Full payment gateway integration (abstracted, mocked).
- Advanced logistics and shipping provider integrations.
- Deep analytics visualizations and cohort reporting.
